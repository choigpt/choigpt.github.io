- **Level 1** — 경우의 수로 이해하는 조건부 확률. 주사위와 카드.

- **Level 3** — 직관이 무너지는 문제들과 실전 응용. 몬티 홀, 심슨의 역설, 나이브 베이즈.

# Level 1: 경우의 수에서 출발

## 확률이란

주사위를 던져서 짝수가 나올 확률:

$$P(\text{짝수}) = \frac{\text{짝수인 경우의 수}}{\text{전체 경우의 수}} = \frac{3}{6} = \frac{1}{2}$$

분모는 **전체 가능한 경우**, 분자는 **원하는 경우**. 이것만 알면 된다.

## 조건이 붙으면 — 전체가 줄어든다

"홀수가 나왔다"는 정보를 알게 되었다고 하자. 그러면 전체 경우의 수가 바뀐다.

```
원래 전체: {1, 2, 3, 4, 5, 6}  ← 6가지

"홀수가 나왔다" → 남은 경우: {1, 3, 5}  ← 3가지로 줄어듦!

이 중에서 "3 이상"인 것: {3, 5}  ← 2가지

P(3 이상 | 홀수) = 2/3
```

이것이 **조건부 확률**의 전부다. 조건이 붙으면 전체가 줄어들고, 줄어든 전체 안에서 비율을 다시 계산한다.

그림으로 보면:

```
전체 6가지:
  ┌─────────────────────────────────┐
  │  1    2    3    4    5    6     │
  │            ├─ 3이상 ──────┤     │
  │  ├홀수┤    ├홀수┤    ├홀수┤     │
  └─────────────────────────────────┘

"홀수" 조건 적용 → 2, 4, 6이 사라진다:
  ┌─────────────────┐
  │  1    3    5     │  ← 이것이 새로운 전체
  │       ├3이상┤    │
  └─────────────────┘

새로운 전체 3가지 중 "3 이상"은 2가지 → 2/3
```

## 공식

위의 직관을 공식으로 쓰면:

$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

- $$P(B)$$: 조건(새로운 전체)의 확률
- $$P(A \cap B)$$: A와 B가 동시에 일어나는 확률
- $$P(A \mid B)$$: B가 일어난 세계 안에서 A의 비율

주사위 예제에 대입:

$$P(\text{3이상} \mid \text{홀수}) = \frac{P(\{3,5\})}{P(\{1,3,5\})} = \frac{2/6}{3/6} = \frac{2}{3}$$

분자분모의 6이 약분되므로, 결국 "경우의 수의 비율"과 같다.

## 예제: 카드

52장에서 한 장을 뽑았다. 그림 카드(J, Q, K)라는 것을 알게 되었다. 이것이 킹(K)일 확률은?

```
전체: 52장

"그림 카드" 조건 적용 → J, Q, K만 남음: 12장 (새로운 전체)

그 중 K: 4장

P(K | 그림 카드) = 4/12 = 1/3
```

J, Q, K 세 종류 중 하나이므로 $$\frac{1}{3}$$. 직관과 정확히 맞는다.

## 예제: 학급

30명의 학급에서:
- 안경 쓴 학생: 12명
- 여학생: 18명
- 안경 쓴 여학생: 8명

여학생 중 안경 쓴 학생의 비율은?

```
전체: 30명

"여학생" 조건 적용 → 18명 (새로운 전체)

그 중 안경: 8명

P(안경 | 여학생) = 8/18 ≈ 44.4%
```

참고로 전체 학생 중 안경 비율은 12/30 = 40%. 조건을 붙이면 비율이 달라진다.

# Level 2: 공식의 의미를 잡는다

## 곱셈 규칙 — 교집합을 분해한다

조건부 확률 공식의 양변에 $$P(B)$$를 곱하면:

$$P(A \cap B) = P(A \mid B) \cdot P(B)$$

A와 B의 역할을 바꾸면:

$$P(A \cap B) = P(B \mid A) \cdot P(A)$$

이것이 **곱셈 규칙**이다. "교집합을 직접 구하기 어려울 때, 조건부 확률과 주변 확률의 곱으로 쪼갤 수 있다."

### 예제: 카드 2장 연속 뽑기

52장에서 카드를 2장 연속으로 뽑는다(비복원). 둘 다 에이스일 확률은?

```
직접 구하기: 52장에서 에이스 2장을 뽑는 조합
  → C(4,2) / C(52,2) = 6 / 1326 ≈ 0.0045

곱셈 규칙으로:
  P(1번째 에이스) × P(2번째 에이스 | 1번째 에이스)
  = 4/52 × 3/51
  = 12/2652 ≈ 0.0045  ✓ 같다!
```

곱셈 규칙은 트리 다이어그램으로 시각화하면 자연스럽다:

```
첫 번째                 두 번째                  확률

                     ┌─ 에이스 (3/51)  → 4/52 × 3/51 = 0.0045
     ┌─ 에이스 (4/52)┤
     │               └─ 아님  (48/51) → 4/52 × 48/51 = 0.0724
전체─┤
     │               ┌─ 에이스 (4/51)  → 48/52 × 4/51 = 0.0724
     └─ 아님  (48/52)┤
                     └─ 아님  (47/51) → 48/52 × 47/51 = 0.8507

검증: 0.0045 + 0.0724 + 0.0724 + 0.8507 = 1.0000 ✓
```

가지를 따라가며 확률을 **곱한다**. 모든 끝의 확률을 더하면 1이 된다.

### 3개 이상으로 확장: 연쇄 규칙 (Chain Rule)

$$P(A \cap B \cap C) = P(A) \cdot P(B \mid A) \cdot P(C \mid A \cap B)$$

**예제: 하트 3장 연속 뽑기**

```
P(하트 3장)
  = P(1번째 하트) × P(2번째 하트 | 1번째 하트) × P(3번째 하트 | 앞의 2장 하트)
  = 13/52 × 12/51 × 11/50
  = 1/4 × 12/51 × 11/50
  = 1716/132600
  ≈ 1.3%

각 단계가 이전 결과에 의존한다:
  1번째: 52장 중 하트 13장
  2번째: 51장 중 하트 12장 (1장 빠짐)
  3번째: 50장 중 하트 11장 (2장 빠짐)
```

## 독립 — 조건이 아무 영향을 주지 않는 경우

"홀수가 나왔다"는 정보를 알아도 "3 이상"의 확률이 변하지 않으면, 두 사건은 **독립**이다.

$$P(A \mid B) = P(A)$$

이를 곱셈 규칙에 넣으면:

$$P(A \cap B) = P(A) \cdot P(B)$$

**예제: 동전 두 번**

```
첫 번째 앞면(A), 두 번째 앞면(B)

P(A) = 1/2
P(B) = 1/2
P(A ∩ B) = P(둘 다 앞면) = 1/4

P(A) × P(B) = 1/2 × 1/2 = 1/4 = P(A ∩ B)  ✓ 독립!
```

첫 번째 결과를 알아도 두 번째에 아무 영향이 없다.

**반대로, 카드 비복원 추출은 독립이 아니다:**

```
P(1번째 에이스) = 4/52
P(2번째 에이스) = 4/52  (무조건적으로는 같지만)
P(2번째 에이스 | 1번째 에이스) = 3/51 ≠ 4/52

1번째 결과가 2번째 확률을 바꾸므로 독립이 아니다.
```

### 독립과 배반은 정반대다

이 두 개념은 자주 혼동되지만 **전혀 다르다**. 오히려 반대에 가깝다.

```
배반(mutually exclusive):  동시에 일어날 수 없다
  A ∩ B = ∅
  P(A ∩ B) = 0
  → A가 일어나면 B는 절대 불가능
  → P(B|A) = 0  ← 극단적인 영향!

독립(independent):  서로 영향이 없다
  P(A ∩ B) = P(A) · P(B)
  → A가 일어나든 말든 B의 확률은 동일
  → P(B|A) = P(B)  ← 영향 없음
```

구체적으로:

```
주사위를 던진다.
  A = {1, 2, 3}  (3 이하)
  B = {4, 5, 6}  (4 이상)

배반인가?
  A ∩ B = ∅  → YES (겹치는 게 없다)

독립인가?
  P(A) × P(B) = 1/2 × 1/2 = 1/4
  P(A ∩ B) = 0
  0 ≠ 1/4  → NO!

"3 이하가 나왔다"를 알면 "4 이상"은 확률 0.
극단적인 영향이므로, 배반이면 독립이 아니다.
```

**독립이 되려면 오히려 겹침이 있어야 한다:**

```
  A = {1, 2, 3, 4}  (4 이하)
  B = {1, 3, 5}     (홀수)

  A ∩ B = {1, 3}    ← 겹침이 있다

  P(A) × P(B) = 4/6 × 3/6 = 12/36 = 2/6
  P(A ∩ B) = 2/6

  2/6 = 2/6  ✓ 독립!
```

## 전확률 정리 — P(A)를 조건별로 쪼갠다

P(A)를 직접 구하기 어렵지만, 여러 조건 아래에서의 P(A)는 알 수 있을 때 쓴다.

$$P(A) = P(A \mid B_1) \cdot P(B_1) + P(A \mid B_2) \cdot P(B_2) + \cdots$$

**예제: 공장의 불량률**

공장 3곳에서 부품을 납품받는다:
- 공장 A: 전체의 50%, 불량률 2%
- 공장 B: 전체의 30%, 불량률 3%
- 공장 C: 전체의 20%, 불량률 5%

임의로 부품 하나를 꺼냈을 때 불량일 확률은?

```
직접 구하기는 어렵다. 하지만 각 공장 조건 아래에서의 불량률은 안다.

P(불량) = P(불량|A)·P(A) + P(불량|B)·P(B) + P(불량|C)·P(C)
        = 0.02 × 0.50 + 0.03 × 0.30 + 0.05 × 0.20
        = 0.010 + 0.009 + 0.010
        = 0.029

→ 전체 불량률은 2.9%
```

이것을 **트리 다이어그램**으로 그리면:

```
          ┌─ 불량 (0.02) → 0.50 × 0.02 = 0.010
  ┌─ A (0.50)─┤
  │       └─ 정상 (0.98) → 0.50 × 0.98 = 0.490
  │
  │       ┌─ 불량 (0.03) → 0.30 × 0.03 = 0.009
  ├─ B (0.30)─┤
  │       └─ 정상 (0.97) → 0.30 × 0.97 = 0.291
  │
  │       ┌─ 불량 (0.05) → 0.20 × 0.05 = 0.010
  └─ C (0.20)─┤
          └─ 정상 (0.95) → 0.20 × 0.95 = 0.190

P(불량) = 불량으로 끝나는 가지의 합 = 0.010 + 0.009 + 0.010 = 0.029
```

여기서 한 발 더: "불량품이 나왔는데, 공장 C에서 온 것일 확률은?"

$$P(C \mid \text{불량}) = \frac{P(\text{불량} \mid C) \cdot P(C)}{P(\text{불량})} = \frac{0.05 \times 0.20}{0.029} = \frac{0.010}{0.029} \approx 34.5\%$$

전확률 정리로 분모를 구하고, 곱셈 규칙으로 분자를 구한다. 이것이 [베이즈 정리](/bayes-theorem/)다. 전확률 정리는 베이즈 정리의 분모를 만드는 도구다.

# Level 3: 직관이 무너지는 문제들

여기서부터는 "당연히 이렇겠지"라는 직관이 틀리는 상황을 다룬다. 조건부 확률을 정확히 이해하지 않으면 빠지는 함정들이다.

## 의료 검진 — 99% 정확도의 함정

어떤 질병의 유병률이 1%이고, 검사의 정확도:
- 환자를 양성으로 판정할 확률(민감도) = 99%
- 건강한 사람을 음성으로 판정할 확률(특이도) = 95%

**양성 판정을 받았을 때 실제로 병에 걸렸을 확률은?**

대부분 "99%에 가깝다"고 답한다. 10,000명으로 직접 세어보자.

```
10,000명 검사:

실제 환자:    10,000 × 0.01 = 100명
건강한 사람:  10,000 × 0.99 = 9,900명

환자 100명 → 양성 판정: 100 × 0.99 = 99명   (진양성)
                음성 판정: 100 × 0.01 = 1명    (위음성)

건강 9,900명 → 양성 판정: 9,900 × 0.05 = 495명  (위양성!)
                 음성 판정: 9,900 × 0.95 = 9,405명 (진음성)

양성 판정 전체: 99 + 495 = 594명
그 중 실제 환자: 99명

P(환자 | 양성) = 99 / 594 ≈ 16.7%
```

**99% 정확도인데 양성이어도 실제 확률은 17%에 불과하다.**

왜?

```
핵심: 건강한 사람이 환자보다 99배 많다

건강한 9,900명의 5% 오진 = 495명  ← 절대 수가 많다!
환자 100명의 99% 정확   = 99명    ← 절대 수가 적다

위양성(495)이 진양성(99)의 5배.
```

이것이 **기저율 무시(base rate neglect)**다. "검사 정확도 99%"에 집중하고 "유병률 1%"를 무시하면 틀린다.

**유병률이 달라지면:**

```
같은 검사(민감도 99%, 특이도 95%)로:

유병률    P(환자|양성)    직관과의 괴리
─────────────────────────────────────
0.1%       2.0%          검사 의미 거의 없음
1%        16.7%          ← 위의 예
5%        51.0%          반반
10%       68.8%          꽤 신뢰 가능
50%       95.2%          거의 확실

유병률이 높을수록 양성 결과의 신뢰도가 올라간다.
→ "고위험군부터 검사한다"는 의료 전략의 통계적 근거
```

## 몬티 홀 문제 — 진행자의 정보가 확률을 바꾼다

TV 게임쇼에 3개의 문이 있다. 하나 뒤에 자동차, 나머지 둘 뒤에 염소.

```
1. 참가자가 문 1을 선택한다.
2. 진행자(몬티)가 나머지 문 중 염소가 있는 문을 하나 연다.
   (진행자는 자동차 위치를 안다!)
   예: 문 3을 열어 염소를 보여줌.
3. "문 2로 바꾸시겠습니까?"
```

직관: "남은 문이 2개니까 반반 아닌가?"

**모든 경우를 나열해보자:**

```
자동차 위치 | 참가자 선택 | 진행자가 여는 문 | 바꾸면?  | 안 바꾸면?
──────────────────────────────────────────────────────────
문 1        | 문 1       | 문 2 또는 문 3   | 염소     | 자동차
문 2        | 문 1       | 문 3 (문 2는 못 열음) | 자동차   | 염소
문 3        | 문 1       | 문 2 (문 3은 못 열음) | 자동차   | 염소

각 경우의 확률은 1/3.

바꾸면 이기는 경우: 2가지 (문 2, 문 3에 자동차) → 확률 2/3
안 바꾸면 이기는 경우: 1가지 (문 1에 자동차)    → 확률 1/3
```

**바꾸면 이길 확률이 2배다.**

왜 반반이 아닌가? 핵심은 **진행자가 정보를 갖고 행동한다**는 것이다.

```
진행자는 자동차 위치를 알고, 절대 자동차가 있는 문을 열지 않는다.
이 "조건부 행동"이 남은 문의 확률을 바꾼다.

참가자가 처음 고른 문: 1/3 확률로 자동차
나머지 두 문에 자동차가 있을 확률: 2/3
진행자가 염소 문을 열어줌 → 2/3의 확률이 남은 한 문에 집중!
```

만약 진행자가 자동차 위치를 모르고 무작위로 열었다면(우연히 염소가 나왔다면), 그때는 정말로 $$\frac{1}{2}$$이 된다. "누가 어떤 정보를 갖고 행동하는가"에 따라 조건부 확률이 달라진다.

## 심슨의 역설 — 부분과 전체가 뒤집힌다

**부분에서는 A가 B보다 낫는데, 전체에서는 B가 A보다 나은 현상.**

**예: 타율 비교**

```
전반기:
  선수 A: 40타수 12안타 → 타율 .300
  선수 B: 100타수 25안타 → 타율 .250
  → 전반기에는 A가 더 높다

후반기:
  선수 A: 100타수 30안타 → 타율 .300
  선수 B: 40타수 10안타 → 타율 .250
  → 후반기에도 A가 더 높다

시즌 전체:
  선수 A: 140타수 42안타 → 타율 .300
  선수 B: 140타수 35안타 → 타율 .250
  → 전체에서도 A가 높다 (이 경우는 역설이 안 일어남)
```

하지만 타수 비율이 다르면:

```
전반기:
  선수 A: 10타수 9안타 → 타율 .900    ← A가 높다
  선수 B: 100타수 80안타 → 타율 .800

후반기:
  선수 A: 100타수 20안타 → 타율 .200   ← A가 높다
  선수 B: 10타수 1안타 → 타율 .100

시즌 전체:
  선수 A: 110타수 29안타 → 타율 .264
  선수 B: 110타수 81안타 → 타율 .736   ← 뒤집혔다! B가 훨씬 높다
```

**전반기도 후반기도 A가 높은데, 전체에서는 B가 압도적으로 높다!**

왜? A는 쉬운 전반기에 적게 치고(10타수), 어려운 후반기에 많이 쳤다(100타수). B는 그 반대다. **각 기간의 가중치(타수)가 다르기 때문에** 전체 합산 시 결과가 뒤집힌다.

**실제 사례: UC 버클리 입학 (1973)**

```
전체 합격률:
  남성: 44% 합격
  여성: 35% 합격
  → 여성 차별?

그러나 학과별로 보면:
  거의 모든 학과에서 여성 합격률이 같거나 더 높았다!

원인: 여성이 경쟁률 높은 학과에 더 많이 지원했다.
  학과 A (합격률 높음): 남성 825명, 여성 108명 지원
  학과 F (합격률 낮음): 남성 191명, 여성 341명 지원

조건(학과)을 무시하고 전체를 보면 잘못된 결론을 내린다.
```

**교훈:** 조건부 확률에서 **조건을 빠뜨리면** 방향이 뒤집힐 수 있다. "전체 확률"보다 "조건부 확률"이 더 정확한 이야기를 들려줄 때가 많다.

## P(A|B) ≠ P(B|A) — 검사의 오류

조건부 확률에서 가장 위험한 혼동은 **방향을 바꾸는 것**이다.

```
P(기침 | 감기) = 0.8     "감기에 걸리면 기침할 확률 80%"
P(감기 | 기침) = ?        "기침을 하면 감기일 확률은?"

이 둘은 같지 않다!
기침의 원인은 감기 외에도 알레르기, 흡연, 먼지 등 많다.
→ P(감기 | 기침)은 0.3 정도일 수 있다.
```

**법정에서의 오류 (Prosecutor's Fallacy):**

```
DNA 일치 확률이 100만 분의 1이다.

검사: "피고인이 무고할 확률은 100만 분의 1입니다"

이것은 틀렸다!
  P(DNA 일치 | 무고) = 1/1,000,000       ← 이것이 주어진 것
  P(무고 | DNA 일치) = ???                 ← 이것은 다른 값

도시 인구 1000만 명이라면:
  무고한 사람 중 DNA 일치: 9,999,999 × 1/1,000,000 ≈ 10명
  범인: 1명
  DNA 일치하는 사람 총: 11명

  P(무고 | DNA 일치) = 10/11 ≈ 91%

→ DNA가 일치해도 무고할 확률이 91%!
```

$$P(A \mid B) \neq P(B \mid A)$$를 혼동하면 무고한 사람을 범인으로 만들 수 있다.

## 생일 문제 — 연쇄 규칙의 힘

n명이 있을 때 적어도 2명의 생일이 같을 확률은?

여사건("모두 다를 확률")을 연쇄 곱셈 규칙으로:

```
P(모두 다름) = 365/365 × 364/365 × 363/365 × ...

2번째 사람: 1번째와 다를 확률 = 364/365
3번째 사람: 1,2번째와 다를 확률 = 363/365
...
```

```
 n명    P(적어도 2명 같음)
──────────────────────────
  5       2.7%
 10      11.7%
 15      25.3%
 20      41.1%
 23      50.7%    ← 23명이면 이미 반반!
 30      70.6%
 40      89.1%
 50      97.0%
 70      99.9%
```

23명이면 50%를 넘는다. 왜? 23명에서 만들 수 있는 쌍의 수가 $$\binom{23}{2} = 253$$개나 되기 때문이다. 365일 대비 253개의 "시도"가 있으니 겹칠 확률이 높다.

# 실전: 나이브 베이즈 스팸 필터

조건부 확률이 기계학습에서 어떻게 쓰이는지, 실제 계산을 따라가보자.

## 학습 데이터

```
스팸 (4통):                          정상 (6통):
  "무료 당첨 축하 무료"                "회의 일정 확인"
  "무료 할인 이벤트"                   "프로젝트 보고서 제출"
  "당첨 축하합니다"                    "점심 약속 확인"
  "무료 쿠폰 할인"                     "코드 리뷰 부탁"
                                       "서버 점검 일정"
                                       "배포 일정 확인"
```

## 1단계: 사전 확률

$$P(\text{스팸}) = 4/10 = 0.4$$

$$P(\text{정상}) = 6/10 = 0.6$$

## 2단계: 단어별 조건부 확률

```
"무료"가 스팸에서 등장: 4통 중 4통  → P("무료"|스팸) ≈ 4/4
"무료"가 정상에서 등장: 6통 중 0통  → P("무료"|정상) ≈ 0/6

문제: 0이 하나라도 있으면 곱셈 시 전체가 0이 된다.
→ 라플라스 스무딩: 모든 카운트에 1을 더한다.

P("무료"|스팸) = (4+1) / (4+10) = 5/14 ≈ 0.357
P("무료"|정상) = (0+1) / (6+10) = 1/16 ≈ 0.063

P("확인"|스팸) = (0+1) / (4+10) = 1/14 ≈ 0.071
P("확인"|정상) = (3+1) / (6+10) = 4/16 = 0.250
```

> 참고: 여기서는 단순화를 위해 문서 단위 출현 여부를 카운트하면서 분모에 어휘 크기 V를 더하는 혼합 방식을 사용했다. 엄밀하게는 Bernoulli NB(문서 단위)라면 분모에 2를, Multinomial NB(단어 빈도)라면 분모에 V를 더하는 것이 정확하다. 핵심 아이디어 — 0 확률을 방지하기 위해 분자와 분모에 작은 값을 더한다 — 는 동일하다.

## 3단계: 새 이메일 분류

새 이메일: "무료 확인"이 들어왔다. 스팸인가 정상인가?

```
베이즈 정리:
  P(스팸 | "무료 확인") ∝ P(스팸) × P("무료"|스팸) × P("확인"|스팸)
                        = 0.4 × 0.357 × 0.071
                        = 0.01014

  P(정상 | "무료 확인") ∝ P(정상) × P("무료"|정상) × P("확인"|정상)
                        = 0.6 × 0.063 × 0.250
                        = 0.00945

정규화 (합이 1이 되도록):
  P(스팸) = 0.01014 / (0.01014 + 0.00945) = 51.8%
  P(정상) = 0.00945 / (0.01014 + 0.00945) = 48.2%

→ 스팸으로 분류 (51.8%)
```

"무료"가 스팸 쪽 확률을 올리고, "확인"이 정상 쪽 확률을 올린다. 두 단어의 영향이 종합되어 최종 판단이 내려진다.

**왜 "나이브(Naive)"인가?** 각 단어가 **조건부 독립**이라고 가정하기 때문이다. "무료"와 "할인"이 함께 나올 확률이 높지만, 이 관계를 무시한다. 순진한 가정이지만 실전에서 놀라울 정도로 잘 작동한다.

## 이 글에서 가져가야 할 것

```
Level 1에서:
  조건부 확률 = 줄어든 전체 안에서의 비율
  P(A|B) = P(A∩B) / P(B)

Level 2에서:
  곱셈 규칙으로 교집합을 분해할 수 있다
  배반 ≠ 독립 (오히려 정반대)
  전확률 정리 = 조건별 확률의 가중 합 → 베이즈 분모

Level 3에서:
  사전 확률(기저율)이 결과를 지배한다 (의료 검진)
  정보를 가진 행동이 확률을 바꾼다 (몬티 홀)
  조건을 빠뜨리면 결론이 뒤집힌다 (심슨의 역설)
  P(A|B) ≠ P(B|A) (검사의 오류)
```

곱셈 규칙의 두 가지 형태를 등호로 연결하면 [베이즈 정리](/bayes-theorem/)가 나온다.
