## 확률 변수란?

확률 변수(Random Variable)는 **확률 실험의 결과를 실수(real number)에 대응시키는 함수**다.

"변수"라는 이름 때문에 헷갈리기 쉽지만, **확률 변수는 함수다**. 이것이 가장 중요한 포인트다.

### 왜 "변수"가 아니라 "함수"인가?

일반적인 변수와 달리, 확률 변수는:
- **입력**: 확률 실험의 결과 (표본공간의 원소)
- **출력**: 실수 값
- **규칙**: 각 실험 결과를 특정 실수에 대응

이는 명백히 함수의 정의에 해당한다.

```
확률 변수 X: Ω → ℝ
(표본공간) → (실수)
```

## 함수로서의 확률 변수

확률 변수는 함수이므로, 수학적 함수가 만족해야 하는 두 가지 조건을 반드시 따라야 한다.

### 1. 전정의성 (Well-defined)

**정의역의 모든 원소에 대해 함수값이 정의되어야 한다.**

```
표본공간 Ω의 모든 결과 ω에 대해 X(ω)가 정의되어야 함
```

**예시:**

주사위 던지기에서:
```
표본공간 Ω = {1, 2, 3, 4, 5, 6}

✅ 올바른 확률 변수:
X(1) = 1
X(2) = 2
X(3) = 3
X(4) = 4
X(5) = 5
X(6) = 6
→ 모든 결과에 대해 정의됨

❌ 잘못된 확률 변수:
X(1) = 1
X(2) = 2
X(3) = 3
X(4) = 정의되지 않음  ← 문제!
X(5) = 5
X(6) = 6
→ 4가 나왔을 때 대응되는 값이 없음
```

### 2. 유일성 (Single-valued)

**하나의 입력값에는 오직 하나의 출력값만 대응되어야 한다.**

```
같은 결과 ω에 대해 X(ω)는 항상 같은 값이어야 함
```

**예시:**

동전 던지기에서:
```
표본공간 Ω = {앞면, 뒷면}

✅ 올바른 확률 변수:
X(앞면) = 1
X(뒷면) = 0
→ 각 결과가 하나의 값에만 대응

❌ 잘못된 확률 변수:
X(앞면) = 1 또는 2  ← 문제!
X(뒷면) = 0
→ 앞면이 나왔을 때 1인지 2인지 알 수 없음
```

## 확률 변수의 구성 요소

### 1. 표본공간 (Sample Space, Ω)

확률 실험에서 나올 수 있는 **모든 가능한 결과들의 집합**

**예시: 주사위 던지기**
```
Ω = {1, 2, 3, 4, 5, 6}
```

이것이 확률 변수의 **정의역(domain)**이 된다.

### 2. 결과 (Outcome, ω)

실제로 일어난 **하나의 실험 결과**

**예시:**
```
주사위를 던졌더니 4가 나옴
→ ω = 4
```

ω는 표본공간 Ω의 원소다 (ω ∈ Ω).

### 3. 사건 (Event, A)

특정 조건을 만족하는 **결과들의 집합**, 즉 표본공간의 부분집합

**예시:**
```
A = {2, 4, 6}  ← "짝수가 나오는 사건"

이 사건은:
- ω = 2일 때 발생
- ω = 4일 때 발생
- ω = 6일 때 발생
```

사건은 확률을 계산하는 단위가 된다:
```
P(A) = P(짝수) = 3/6 = 1/2
```

### 4. 확률 변수 (Random Variable, X)

표본공간의 결과를 **실수에 대응시키는 함수**

**가장 간단한 예시:**
```
X(ω) = ω

X(1) = 1
X(2) = 2
X(3) = 3
X(4) = 4
X(5) = 5
X(6) = 6
```

이 경우 주사위의 눈이 그대로 확률 변수의 값이 된다.

## 확률 변수의 실제 예시

### 예시 1: 동전 던지기

```
실험: 동전 한 개 던지기

표본공간:
Ω = {앞면(H), 뒷면(T)}

확률 변수 X (앞면이 나온 횟수):
X(H) = 1
X(T) = 0

확률:
P(X = 1) = P(H) = 1/2
P(X = 0) = P(T) = 1/2
```

### 예시 2: 동전 두 개 던지기

```
실험: 동전 두 개 던지기

표본공간:
Ω = {HH, HT, TH, TT}

확률 변수 X (앞면이 나온 횟수):
X(HH) = 2
X(HT) = 1
X(TH) = 1
X(TT) = 0

확률 분포:
P(X = 0) = P({TT}) = 1/4
P(X = 1) = P({HT, TH}) = 2/4 = 1/2
P(X = 2) = P({HH}) = 1/4
```

**주목할 점:**
- HT와 TH는 서로 다른 결과(ω)지만
- 같은 확률 변수 값 X = 1로 대응됨
- 이것이 가능한 이유는 함수는 여러 입력이 같은 출력을 가질 수 있기 때문 (전사함수는 OK, 단사함수일 필요는 없음)

### 예시 3: 주사위에서 짝수 여부

```
실험: 주사위 던지기

표본공간:
Ω = {1, 2, 3, 4, 5, 6}

확률 변수 X (짝수면 1, 홀수면 0):
X(1) = 0
X(2) = 1
X(3) = 0
X(4) = 1
X(5) = 0
X(6) = 1

확률 분포:
P(X = 0) = P({1, 3, 5}) = 3/6 = 1/2
P(X = 1) = P({2, 4, 6}) = 3/6 = 1/2
```

### 예시 4: 주사위 결과를 변환

```
실험: 주사위 던지기

표본공간:
Ω = {1, 2, 3, 4, 5, 6}

확률 변수 X (주사위 눈 × 10):
X(1) = 10
X(2) = 20
X(3) = 30
X(4) = 40
X(5) = 50
X(6) = 60

확률 분포:
P(X = 10) = 1/6
P(X = 20) = 1/6
...
P(X = 60) = 1/6
```

## 왜 확률 변수가 필요한가?

### 1. 수학적 처리의 용이성

실험 결과가 숫자가 아닐 수 있다:
```
동전: {앞면, 뒷면}
카드: {♠, ♥, ♦, ♣}
날씨: {맑음, 흐림, 비}
```

이런 결과들을 직접 다루기는 어렵다. 확률 변수를 사용하면:
```
동전 → X: {앞면→1, 뒷면→0}
카드 → X: {♠→1, ♥→2, ♦→3, ♣→4}
날씨 → X: {맑음→2, 흐림→1, 비→0}
```

이제 **기댓값, 분산 등 수치 계산**이 가능해진다.

### 2. 통일된 분석 도구

확률 변수를 사용하면 서로 다른 실험도 같은 방식으로 분석할 수 있다:

```
실험 1: 동전 10번 던지기에서 앞면 횟수
실험 2: 10명의 환자 중 완치된 사람 수
실험 3: 10개의 제품 중 불량품 수

→ 모두 "이항분포"라는 같은 확률분포로 분석 가능
```

### 3. 복잡한 사건의 표현

```
주사위 실험에서:

"짝수가 나오는 사건" = {2, 4, 6}
→ 확률 변수로 표현: {X는 짝수}
→ 또는: {X ∈ {2, 4, 6}}

"4 이상이 나오는 사건" = {4, 5, 6}
→ 확률 변수로 표현: {X ≥ 4}

"2 이상 5 이하" = {2, 3, 4, 5}
→ 확률 변수로 표현: {2 ≤ X ≤ 5}
```

확률 변수를 사용하면 **부등식으로 사건을 표현**할 수 있어 훨씬 직관적이다.

## 확률 변수의 종류

### 이산 확률 변수 (Discrete Random Variable)

**취할 수 있는 값이 셀 수 있는 (countable) 경우**

**예시:**
```
- 동전 던지기에서 앞면 횟수: 0, 1, 2, 3, ...
- 주사위 눈: 1, 2, 3, 4, 5, 6
- 하루에 받은 이메일 개수: 0, 1, 2, 3, ...
- 1년 동안 교통사고 횟수: 0, 1, 2, 3, ...
```

특징:
- 값과 값 사이에 "빈 공간"이 있음
- 확률질량함수(PMF)로 표현
- P(X = k) 형태로 확률 계산

### 연속 확률 변수 (Continuous Random Variable)

**취할 수 있는 값이 연속적인 (continuous) 경우**

**예시:**
```
- 사람의 키: 170.5cm, 170.523cm, ...
- 시험 시간: 25.3분, 25.37분, ...
- 온도: 23.5°C, 23.57°C, ...
- 제품의 무게: 100.2g, 100.23g, ...
```

특징:
- 값과 값 사이에 무한히 많은 값이 존재
- 확률밀도함수(PDF)로 표현
- P(a ≤ X ≤ b) 형태로 확률 계산
- P(X = k) = 0 (특정 값일 확률은 0)

## 핵심 정리

**1. 확률 변수는 함수다**
```
입력: 실험 결과 (ω ∈ Ω)
출력: 실수 (X(ω) ∈ ℝ)
```

**2. 함수의 조건을 만족해야 한다**
```
- 전정의성: 모든 결과에 대해 정의
- 유일성: 하나의 결과는 하나의 값에만 대응
```

**3. 확률 변수의 목적**
```
- 비수치적 결과를 수치로 변환
- 수학적 분석 가능하게 만듦
- 기댓값, 분산 등 통계량 계산
- 통일된 프레임워크 제공
```

**4. 표기법**
```
표본공간: Ω (오메가 대문자)
결과: ω (오메가 소문자)
사건: A, B, C, ...
확률 변수: X, Y, Z, ...
확률: P(·)
```

## 실생활 예시로 마무리

**예시: 복권 당첨**

```
실험: 복권 한 장 구매

표본공간:
Ω = {1등, 2등, 3등, 4등, 5등, 꽝}

확률 변수 X (당첨금액, 원):
X(1등) = 10,000,000
X(2등) = 1,000,000
X(3등) = 100,000
X(4등) = 10,000
X(5등) = 1,000
X(꽝) = 0

기댓값 (평균 당첨금):
E[X] = 10,000,000 × P(1등) + 1,000,000 × P(2등) + ...

→ 복권을 많이 사면 평균적으로 이만큼 받을 것으로 기대
→ 기댓값이 복권 가격보다 낮으면 장기적으로는 손해
```

이처럼 확률 변수는 **불확실한 결과를 수치화**하여 **합리적인 의사결정**을 돕는 도구다.
